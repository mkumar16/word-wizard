# Word Wizard 🎮

A fun, educational spelling game for kids! Tap letters to fill slots, match emoji hints, and learn new vocabulary with colorful visuals.

## How to Play

1. **Tap letters** to select them, then **tap a slot** to place them
2. **Match the emoji hint** to guess the word
3. **Use hints** if you get stuck (💡 button)
4. **Build streaks** for bonus points! 🔥
5. **Choose a category** from 15 options
6. **Complete all 5 levels** to master a category

## Features

- 📚 **15 Categories**: General, Colors, Numbers, Fruits, Vegetables, Animals, Birds, Vehicles, Body Parts, Clothing, Weather, Food, Countries, Shapes, Toys
- 🎯 **5 Difficulty Levels** per category (3-7+ letter words)
- 🐱 Emoji hints for each word
- ⭐ Scoring system with streak bonuses
- 🎉 Confetti celebrations
- 📱 **Web & Mobile** (React Native)
- 🔊 Sound effects (toggle on/off)
- 💾 Progress saved automatically
