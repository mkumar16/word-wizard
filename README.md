# Word Wizard 🎮

A fun, educational spelling game for kids! Drag and drop letters to form words and learn new vocabulary with colorful visuals.

## How to Play

1. **Drag letters** from the letter bank to the empty slots
2. **Match the emoji hint** to guess the word
3. **Use hints** if you get stuck (💡 button)
4. **Build streaks** for bonus points! 🔥

## Features

- 🎯 5 levels with 25 words
- 🐱 Emoji hints for each word
- ⭐ Scoring system with streak bonuses
- 🎉 Confetti celebrations
- 📱 Mobile-friendly (touch support)
- 🔊 Sound effects (toggle on/off)
- 💾 Progress saved automatically
